# SentinelOps AI

Enterprise-grade asset intelligence website.

## Run locally (dev server)

```bash
npm install
npm run dev
```

Then open: **http://127.0.0.1:5173**

## Open website on Windows (no dev server)

1. Run **`npm install`** once in this folder (if you haven’t already).
2. Double‑click **`Open website.bat`** in the project folder.
3. The site will build, a preview server will start, and your browser will open to **http://127.0.0.1:4173**.

Close the “SentinelOps AI Preview” command window when you’re done to stop the server.

## Build only

```bash
npm run build
npm run preview
```

Preview serves at **http://127.0.0.1:4173**.
