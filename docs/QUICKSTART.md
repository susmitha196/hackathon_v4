## Quick Commands

**Run Streamlit App:**
```bash
streamlit run src/app.py
```

Or use the convenience script:
```bash
python run_app.py
```

**Run FastAPI Backend:**
```bash
cd src && python api.py
```

Or use the convenience script:
```bash
python run_api.py
```

**Test n8n Webhook:**
```bash
python scripts/test_n8n_webhook.py
```

For detailed instructions, see [docs/QUICKSTART.md](docs/QUICKSTART.md).
